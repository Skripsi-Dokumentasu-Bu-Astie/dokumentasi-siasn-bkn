# Breadcrumb Monitoring Paraf Pertek


Breadcrumb merupakan navigasi kecil yang berada dibagian atas
nama menu pada halaman sebuah website. Breadcrumb berfungsi untuk membantu user
mengetahui posisinya dalam situs web.


```Berikut merupakan tampilan dari Breadcrumb Menu Monitoring Paraf Pertek:```
![Breadcrumb](../../src/img/monitoring-pp/M_BreadcrumbMPP.png)

`BreadcrumbMPP` memiliki struktur komponen sebagai berikut:

| Nama Komponen         | Contoh Pemanggilan <br/> Komponen                  | Properti/Atribut   | Tipe Data <br/> Atribut | Penjelasan
