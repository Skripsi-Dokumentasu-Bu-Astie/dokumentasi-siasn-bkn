# Teble Monitoring Paraf Pertek

Table Monitoring Paraf Pertek (TableMPP) merupakan tampilan tabel yang menampilkan data monitoring untuk
paraf pertek sebagai salah satu syarat usulan kenaikan pangkat untuk ASN. Pada tabel ini akan ditampilkan
data yang lebih spesifikasi setelah user melakukan pencarian data usulan.


```Berikut merupakan tampilan dari Table Menu Monitoring Paraf Pertek:```

![Table MPP](../../src/img/monitoring-pp/TableMPP.png)
