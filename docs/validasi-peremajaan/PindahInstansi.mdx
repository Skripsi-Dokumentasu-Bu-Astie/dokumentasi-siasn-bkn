# Pindah Instansi

Pindah Instansi merupakan komponen yang berisi informasi tentang jenis pemindahan,
jenis pegawai, jenis jabatan lama, jenis jabatan baru, Instansi kerja lama,
Instansi kerja baru, satuan kerja lama, satuan kerja baru, unor lama unor baru,
jabatan fungsional lama, jabatan fungsional baru, instansi induk lama, instansi induk baru,
satuan kerja induk lama, satuan kerja induk baru, lokasi kerja lama, lokasi kerja baru, KPPN baru,
jabatan fungsional umum baru, nomor surat instansi asal dan nomor surat instansi tujuan beserta tanggal suratnya, nomor SK BKN beserta tanggal suratnya,
dan TMT PI.

### Struktur Komponen

`PindahInstansi` memiliki struktur komponen sebagai berikut:

| Nama Komponen  | Contoh Pemanggilan <br/> Komponen | Properti/Atribut | Tipe Data <br/> Atribut | Penjelasan                                                                                                  |
| -------------- | --------------------------------- | ---------------- | ----------------------- | ----------------------------------------------------------------------------------------------------------- |
| PindahInstansi | `<PindahInstansi />`              | -                | -                       | Komponen PindahInstansi berisi informasi <br/> data ASN, jenis layanan, <br/> hingga dokumen-dokumen usulan |
