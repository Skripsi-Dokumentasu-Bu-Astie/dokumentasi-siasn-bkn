# Detail Monitoring Usulan

Detail Monitoring Usulan merupakan komponen halaman yang digunakan untuk
melihat detail lengkap dari usulan yang di ajukan sebelumnya.

![Detail Monitoring Usulan](../../src/img/DetailMonitoringUsulan.png)

### Struktur Komponen

`DetailMonitoringUsulan` memiliki struktur komponen sebagai berikut:

| Nama Komponen          | Contoh Pemanggilan <br/> Komponen                                                                                   | Properti/Atribut | Tipe Data <br/> Atribut | Penjelasan                                                                                                                                                                                              |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------- | ---------------- | ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| DetailMonitoringUsulan | `<DetailMonitoringUsulan`<br/>&nbsp;&nbsp;`idUsul={id}`<br/>&nbsp;&nbsp;`dataUsulan={dataRes}`<br/>&nbsp;&nbsp;`/>` | `idUsul`         | `String`                | Properti id usul berisi informasi <br/> id usulan yang diajukan yang akan digunakan <br/> sebagai acuan untuk mengambil <br/> data Dokumen Persetujuan Teknis <br/> berdasarkan id usulan yang diajukan |
|                        |                                                                                                                     | `dataUsulan`     | `Object`                | Properti data usulan berisi informasi <br/> data diri ASN, jenis layanan, <br/> hingga dokumen-dokumen usulan                                                                                           |
