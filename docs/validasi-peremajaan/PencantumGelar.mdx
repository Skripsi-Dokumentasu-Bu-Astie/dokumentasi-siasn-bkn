# Pencantum Gelar

Pencantum merupakan komponen yang Gelar berisi informasi tentang tingkat pendidikan ASN,
pendidikan terakhir ASN,
tanggal lulus, tahun lulus, nomor ijazah, nama sekolah, gelar depan, gelar belakang, dan
Nama jabatan kepala biro kepegawaian/BKPSDM.

![Pencantuman Gelar](../../src/img/PencantumanGelar.png)

### Struktur Komponen

`PencantumanGelar` memiliki struktur komponen sebagai berikut:

| Nama Komponen    | Contoh Pemanggilan <br/> Komponen                                                                                          | Properti/Atribut | Tipe Data <br/> Atribut | Penjelasan                                                                                                    |
| ---------------- | -------------------------------------------------------------------------------------------------------------------------- | ---------------- | ----------------------- | ------------------------------------------------------------------------------------------------------------- |
| PencantumanGelar | `<PencantumanGelar` <br/> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;`dataUsulan=` <br/> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;`{dataUsulan}/>` | `dataUsulan`     | `Object`                | Properti data usulan berisi informasi <br/> data diri ASN, jenis layanan, <br/> hingga dokumen-dokumen usulan |
