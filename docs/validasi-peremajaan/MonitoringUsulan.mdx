# Monitoring Usulan

Monitoring Usulan merupakan komponen halaman yang berisi
semua data yang diajukan sebelumnya yang dikemas dalam sebuah
tabel serta fitur pencarian usulan bedasarkan beberapa data usulan.

![Monitoring Usulan](../../src/img/MonitoringUsulan.png)

### Struktur Komponen

`MonitoringUsulan` memiliki struktur komponen sebagai berikut:

| Nama Komponen    | Contoh Pemanggilan <br/> Komponen | Properti/Atribut | Tipe Data <br/> Atribut | Penjelasan                                                                                                       |
| ---------------- | --------------------------------- | ---------------- | ----------------------- | ---------------------------------------------------------------------------------------------------------------- |
| MonitoringUsulan | `<MonitoringUsulan />`            | -                | -                       | Pada MonitoringUsulan berisi informasi <br/> semua usulan yang sudah diajukan <br/> serta fitur pencarian usulan |
