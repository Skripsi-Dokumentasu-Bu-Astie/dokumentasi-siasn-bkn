# Profesi

Profesi merupakan komponen yang berisi informasi tentang golongan ASN,
pendidikan terakhir ASN,
jabatan, masa kerja, serta lembaga tempat ASN
bekerja.

### Struktur Komponen

`Profesi` memiliki struktur komponen sebagai berikut:

| Nama Komponen | Contoh Pemanggilan <br/> Komponen | Properti/Atribut | Tipe Data <br/> Atribut | Penjelasan                                                                                                |
| ------------- | --------------------------------- | ---------------- | ----------------------- | --------------------------------------------------------------------------------------------------------- |
| Profesi       | `<Profesi />`                     | -                | -                       | Pada komponen Profesi berisi informasi <br/> data ASN, jenis layanan, <br/> hingga dokumen-dokumen usulan |
