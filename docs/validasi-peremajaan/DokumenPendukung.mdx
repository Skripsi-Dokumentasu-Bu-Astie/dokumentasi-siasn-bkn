# Dokumen Pendukung

Dokumen Pendukung merupakan komponen dan termasuk bagian dari
langkah pertama dalam proses validasi usulan peremajaan.
Bagian Dokumen Pendukung berisi informasi mengenai
dokumen-dokumen milik ASN yang digunakan sebagai
pendukung keputusan dalam validasi usulan peremajaan
yang diajukan.

### Struktur Komponen

`DokPendukung` memiliki struktur komponen sebagai berikut:

| Nama Komponen | Contoh Pemanggilan <br/> Komponen                                                                                                                                                                                                                                                                                                                  | Properti/Atribut   | Tipe Data <br/> Atribut | Penjelasan                                                                                                                                                                                               |
| ------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------ | ----------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| DokPendukung  | `<DokPendukung` <br/> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;`dataIsHere=` <br/> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;`{dataIsHere}` <br/> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;`setDataIsHere=` <br/> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;`{dataIsHere}` <br/> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;`dokumenTerupload=` <br/> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;`{finalArray}` <br/> `/>` | `dataIsHere`       | `Boolean`               | dataIsHere bertipe <br/> boolean yang berguna <br/> untuk memeriksa apakah ASN <br/> yang mengajukan usulan <br/> memiliki dokumen pendukung.                                                            |
|               |                                                                                                                                                                                                                                                                                                                                                    | `dokumenTerupload` | `Object`                | dokumenTerupload <br/> bertipe array of object <br/> yang berisi data <br/> dokumen-dokumen <br/>pendukung yang <br/> dilampirkan oleh ASN <br/> yang melakukan pengajuan <br/> usulan kenaikan pangkat. |
|               |                                                                                                                                                                                                                                                                                                                                                    | `setdataIsHere`    | `Boolean`               | setDataIsHere bertipe <br/> boolean yang berguna <br/> untuk memeriksa apakah ASN <br/> yang mengajukan usulan <br/> memiliki dokumen pendukung.                                                         |
