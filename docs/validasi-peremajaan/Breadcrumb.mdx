# Breadcrumb

Breadcrumb merupakan komponen
yang digunakan untuk menampilkan hierarki
navigasi pada halaman.

![Breadcrumb](../../src/img/Breadcrumb.png)

### Struktur Komponen

`Breadcrumb` memiliki struktur komponen sebagai berikut:

| Nama Komponen | Contoh Pemanggilan <br/> Komponen | Properti/Atribut | Tipe Data <br/> Atribut | Penjelasan                                                                                                              |
| ------------- | --------------------------------- | ---------------- | ----------------------- | ----------------------------------------------------------------------------------------------------------------------- |
| Breadcrumb    | `<Breadcrumb />`                  | -                | -                       | Pada Breadcrumb terdapat <br/> hierarki navigasi halaman <br/> yang memudahkan pengguna <br/> dalam menjelajahi halaman |
