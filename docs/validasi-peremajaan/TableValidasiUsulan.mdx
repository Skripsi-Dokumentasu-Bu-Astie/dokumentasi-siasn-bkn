# Tabel Validasi Usulan

Tabel Validasi Usulan adalah komponen yang berada
pada halaman Validasi Usulan yang berguna untuk menampilkan
data usulan.

![Tabel Validasi Usulan](../../src/img/TabelUsulan.png)

### Struktur Komponen

`TableVU` memiliki struktur komponen sebagai berikut:

| Nama Komponen | Contoh Pemanggilan <br/> Komponen                                                                        | Properti/Atribut | Tipe Data <br/> Atribut | Penjelasan                                                                                           |
| ------------- | -------------------------------------------------------------------------------------------------------- | ---------------- | ----------------------- | ---------------------------------------------------------------------------------------------------- |
| TableVU       | `<TableVU` <br/> &nbsp;&nbsp;&nbsp;&nbsp; `dataTable={data}` <br/> &nbsp;&nbsp;&nbsp;&nbsp; `</TableVU>` | `dataTable`      | `Object`                | Properti dataTable berisi data <br/> validasi usulan peremajaan ASN <br/> yang di tampilkan di table |
