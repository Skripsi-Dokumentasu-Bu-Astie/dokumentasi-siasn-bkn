# Card Informasi ASN

Komponen CardProfile atau Card Informasi ASN berisi informasi singkat ASN yang mengajukan
usulan Peremajaan. Informasi yang ditampilkan yaitu
Nama ASN, NIP, Jenis Prosedur, dan Tanggal Pengajuan.

![Card Info ASN](../../src/img/CardProfile.png)

### Struktur Komponen

`CardProfile` memiliki struktur komponen sebagai berikut:

| Nama Komponen | Contoh Pemanggilan <br/> Komponen                                                                                              | Properti/Atribut | Tipe Data <br/> Atribut | Penjelasan                                                                                                                            |
| ------------- | ------------------------------------------------------------------------------------------------------------------------------ | ---------------- | ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| CardProfile   | `<CardProfile` <br/> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;`dataUsulan=` <br/> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;`{dataUsulan}` <br/> `/>` | `dataUsulan`     | `Object`                | Properti data usulan berisi informasi <br/> data diri ASN, jenis layanan, <br/> Tanggal Pengajuan hingga <br/> dokumen-dokumen usulan |
