# Layout

Layout merupakan *custom component* yang digunakan untuk mengatur tata letak utama dari halaman website.

Layout terdiri dari Header dan Footer.

```Berikut merupakan tampilan dari Layout```
![Layout](../../src/img/pembuatan-SK-Kolektif/Layout.jpeg)

### Struktur Komponen

`Layout` memiliki struktur komponen sebagai berikut:



| Nama Komponen | Contoh Pemanggilan <br/> Komponen                                             | Properti/Atribut     | Tipe Data <br/> Atribut | Penjelasan                                                                 |
| --------------| ------------------------------------------------------------------------------| -------------------- | ----------------------- | -------------------------------------------------------------------------- |
| Layout        | `<Layout>` <br/> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;`{children}` <br/> `</Layout>` | `children component` | `Any`                   | Children Component adalah <br/> komponen yang berada di dalam <br/> Layout |
