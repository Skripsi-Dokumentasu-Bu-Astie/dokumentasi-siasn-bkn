# Table Pembuatan SK Kolektif

Table pembuatan SK kolektif merupakan tabel yang terdapat pada halaman [Tahap 2](/docs-pembuatan-sk-kolektif-step-two "title"), [Tahap 3](/docs-pembuatan-sk-kolektif-step-three "title"), [Tahap 4](/docs-pembuatan-sk-kolektif-step-four "title") yang menampilkan data ASN yang telah dipilih oleh admin SI-ASN BKN untuk dilakukan pembuatan SK kolektif. Tabel ini
berisi data ASN yang telah dipilih pada [Table Surat Usulan](/docs-pembuatan-sk-kolektif-table-step-1 "title").

<br/>
