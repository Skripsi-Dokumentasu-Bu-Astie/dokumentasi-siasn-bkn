# Breadcrumb

Breadcrumb adalah komponen
yang digunakan untuk menunjukkan hirarki
navigasi pada halaman.

![Breadcrumb](../../src/img/pembuatan-SK-Kolektif/Breadcrumb.jpeg)

### Struktur Komponen

`Breadcrumb` memiliki struktur komponen sebagai berikut:

| Nama Komponen | Contoh Pemanggilan <br/> Komponen | Properti/Atribut | Tipe Data <br/> Atribut | Penjelasan                                                                                                             |
| ------------- | --------------------------------- | ---------------- | ----------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| Breadcrumb    | `<Breadcrumb/>`                   | -                | -                       | Pada Breadcrumb terdapat <br/> hirarki navigasi halaman <br/> yang memudahkan pengguna <br/> dalam menjelajahi halaman |
