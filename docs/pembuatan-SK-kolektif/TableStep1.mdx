# Table Surat Usulan

Table Surat Usulan merupakan tabel yang terdapat pada halaman [Tahap 1](/docs-pembuatan-sk-kolektif-step-one "title") dan digunakan untuk menampilkan data ASN yang telah mengajukan surat usulan. Pada table ini Admin SI-ASN BKN dapat memilih ASN yang akan dilakukan pembuatan SK kolektif. 
Table ini terdiri dari beberapa kolom, yaitu:
* NIP
* Nama 
* Jenis Kenaikan
* Jenis Prosedur
* Instansi
* Periode 
* Status Usulan
* Aksi yang terdiri dari lihat resume data pribadi ASN, data atasan pada lembaga tempat ASN yang bersangkutan bekerja, dan preview dari dokumen persetujuan teknis.
