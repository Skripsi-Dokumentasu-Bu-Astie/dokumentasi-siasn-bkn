# Resume Data 2

Resume Data 2 merupakan salah satu modal yang berada pada kolom aksi pada [Tabel Surat Usulan](/docs-pembuatan-sk-kolektif-table-step-1 "title"). 
Resume Data 2 berisi data Kantor Pelayanan Perbendaharaan Negara (KPPN), data atasan pada lembaga tempat ASN yang bersangkutan bekerja, 
dan alasan kenaikan pangkat ASN. 

<br/>

### Component Structure

`Resume Data 2` memiliki struktur komponen sebagai berikut.

| Nama Komponen | Contoh Pemanggilan <br/> Komponen                                                                                                                                               | Properti/Atribut | Tipe Data <br/> Atribut | Penjelasan                                                                                                                        |
| ------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------| ------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| `ResumeData2` | `<ResumeData2` <br/> &nbsp;&nbsp;&nbsp;&nbsp; `dataUsulan=`  <br/> &nbsp;&nbsp;&nbsp;&nbsp; `{props?.dataUsulan}` <br/> &nbsp;&nbsp;&nbsp;&nbsp; `tipeForm={'view'}` <br/> `/>` | `dataUsulan`     | `object`                | `dataUsulan` merupakan atribut <br/> yang berisi informasi data <br/> pribadi ASN, jenis layanan, <br/> dan dokumen-dokumen usulan|
|               |                                                                                                                                                                                 | `tipeForm`       | `string`                | `tipeForm` merupakan atribut yang berisi jenis form yang akan ditampilkan.
