# Step Three Component

`StepThree` merupakan *custom component* yang berisi *value* dari *key* `content` pada  `component fields`. 

#### `StepThree` disusun atas beberapa *section*, yaitu 
* [Form Pilih Pejabat Petikan](/docs-pembuatan-sk-kolektif-form-pilih-pejabat-petikan "title")
* Preview SK Petikan

<br/>

```Berikut merupakan tampilan dari halaman Pembuatan SK Kolektif Tahap 3```

### Component Structure

`StepThree` memiliki struktur komponen sebagai berikut. 

| Nama Komponen | Contoh Pemanggilan <br/> Komponen | Properti/Atribut | Tipe Data <br/> Atribut | Penjelasan |
| ------------- | :---------------------------------| ---------------- | ----------------------- | -----------|
|`StepThree`    | `<StepThree />`                     | -              | -                       | `StepThree` berisi semua <br/> konten yang ada pada <br/> menu halaman Pembuatan <br/> SK Kolektif Tahap 3. |
