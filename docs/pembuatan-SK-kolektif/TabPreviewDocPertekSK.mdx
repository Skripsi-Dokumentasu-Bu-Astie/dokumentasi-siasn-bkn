# Preview Doc Pertek SK

Preview Doc Pertek SK merupakan salah satu modal yang berada pada kolom aksi pada [Tabel Surat Usulan](/docs-pembuatan-sk-kolektif-table-step-1 "title"). 
Preview Doc Pertek SK ini berisi detail dari dokumen pertimbangan teknis yang bisa di download dengan ekstensi pdf.
