# Breadcrumb

`Breadcrumb` adalah komponen
yang digunakan untuk menunjukkan hirarki
navigasi pada halaman. Pada menu Validasi Usulan Kenaikan Pangkat,
[`Breadcrumb`](/validasi-usulan-kenaikan-pangkat/breadcrumb "Breadcrumb")
sering digunakan hampir di setiap halaman seperti
[Halaman Validasi Usulan](/validasi-usulan-kenaikan-pangkat/halaman-validasi-usulan "Halaman Validasi Usulan"),
[Halaman Step-1](/validasi-usulan-kenaikan-pangkat/halaman-step-1 "Halaman Step-1"),
[Halaman Step-2](/validasi-usulan-kenaikan-pangkat/halaman-step-2 "Halaman Step-2"),
serta [Halaman Tolak Usulan](/validasi-usulan-kenaikan-pangkat/halaman-tolak-usulan "Halaman Tolak Usulan").

![Breadcrumb](../../src/img/validasi-usulan-kenaikan-pangkat/Breadcrumb.png)

### Struktur Komponen

`Breadcrumb` memiliki struktur komponen sebagai berikut:

| Nama Komponen | Contoh Pemanggilan <br/> Komponen | Properti/Atribut | Tipe Data <br/> Atribut | Penjelasan                                                                                                             |
| ------------- | --------------------------------- | ---------------- | ----------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| Breadcrumb    | `<Breadcrumb />`                  | -                | -                       | Pada Breadcrumb terdapat <br/> hirarki navigasi halaman <br/> yang memudahkan pengguna <br/> dalam menjelajahi halaman |
