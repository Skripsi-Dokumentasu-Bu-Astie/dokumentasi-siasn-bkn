# Resume Data 2


Resume Data 2 berisi informasi tentang
Kantor Pelayanan Perbendaharaan Negara (KPPN),
Kantor Regional (Konreg) 
atasan pada lembaga tempat ASN yang bersangkutan bekerja, serta
alasan kenaikan pangkat.
