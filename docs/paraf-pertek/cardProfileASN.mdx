# Card Profile Informasi ASN

Card Informasi ASN berisi informasi singkat ASN yang mengajukan
paraf pertek untuk kenaikan pangkat. Informasi yang ditampilkan seperti
Nama ASN, NIP, Instansi Kerja, Jenis Prosedur, dan Tanggal Usul.


```Berikut merupakan tampilan dari Card Profile Menu Paraf Pertek:```
![Card Info ASN](../../src/img/paraf-pertek/PP_CardProfile.png)
