# Card Profile

Card Profile adalah sebuah komponen Card untuk memunculkan detail profile user

### Struktur Komponen

`Card Profile` memiliki struktur Atribut sebagai berikut:

| Library / Modul | Komponen yang digunakan | Properti/Atribut                       | Penjelasan                                                     |
| --------------- | ----------------------- | -------------------------------------- | -------------------------------------------------------------- |
|                 |                         | `-`                                    |                                                                |
|                 |                         | `<className="card card-profile my-3">` | atribut class yang berfungsi untuk memrikan style pada element |
