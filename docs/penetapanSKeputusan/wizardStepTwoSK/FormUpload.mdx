# Form Upload

Form Upload adalah komponen
yang digunakan untuk mengupload, mengganti
file atau gambar pada sebuah form.

```
Komponen ini belum/tidak dipakai pada website.
```

### Struktur Komponen

`FormUpload` memiliki struktur komponen sebagai berikut:

| Library / Modul   | Komponen yang digunakan | Properti/Atribut | Penjelasan                                                                                                |
| ----------------- | ----------------------- | ---------------- | --------------------------------------------------------------------------------------------------------- |
| `axios`           | `axios`                 |                  | `axios` Digunakan untuk mengambil, mengirim data dari/ke server.                                          |
| `react`           | `react`                 |                  | `react` Merupakan library react. Berfungsi supaya dapat mengakses pada semua komponen pada library react. |
| `IconCalendarSU2` |                         |                  | `IconCalendarSU2` Merupakan icon kalender pada tanggal.                                                   |

### Cara Pemanggilan Komponen

| Nama Komponen | Contoh Pemanggilan <br/> Komponen | Penjelasan                                                               |
| ------------- | --------------------------------- | ------------------------------------------------------------------------ |
| `FormUpload`  | `<FormUpload />`                  | `FormUpload` ini digunakan untuk user menginput suatu data pada website. |
