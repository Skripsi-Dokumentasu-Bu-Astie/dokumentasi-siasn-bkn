# Date Picker

Date Picker adalah komponen
yang digunakan untuk menampilkan tanggal
pada langkah kedua di halaman Ajukan Surat Keputusan.

![DatePicker](../../../src/img/WizardSK2/Date.png)

### Struktur Komponen

`DatePicker` memiliki struktur komponen sebagai berikut:

| Library / Modul    | Komponen yang digunakan | Properti/Atribut | Penjelasan                                                                                                        |
| ------------------ | ----------------------- | ---------------- | ----------------------------------------------------------------------------------------------------------------- |
| `react`            | `useState`              |                  | `react` Merupakan library react, <br/> `useState` digunakan untuk menyimpan dan mengatur state data pada halaman. |
| `react-datepicker` |                         |                  | `react-datepicker` Digunakan untuk user memilih tanggal.                                                          |
| `IconCalendarSU2`  |                         |                  | `IconCalendarSU2` Merupakan icon kalender pada tanggal.                                                           |
