# Preview Dok Pertek

Preview Dok Pertek adalah custom component yang digunakan untuk
menampilkan preview surat keputusan pada halaman Ajukan Surat Keputusan.

![IconCalender](../../../src/img/WizardSK2/Preview.png)

### Cara Pemanggilan Komponen

| Nama Komponen      | Contoh Pemanggilan <br/> Komponen | Properti/Atribut | Tipe Data <br/> Atribut | Penjelasan                                                                                                      |
| ------------------ | --------------------------------- | ---------------- | ----------------------- | --------------------------------------------------------------------------------------------------------------- |
| `PreviewDokPertek` | `<PreviewDokPertek />`            | -                | Object                  | `PreviewDokPertek` ini digunakan untuk menampilkan preview surat keputusan pada halaman Ajukan Surat Keputusan. |
