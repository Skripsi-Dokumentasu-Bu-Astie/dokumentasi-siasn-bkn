# Breadcrumb

Breadcrumb adalah komponen
yang digunakan untuk menunjukkan hirarki
navigasi pada halaman.

![Breadcrumb](../../../src/img/detailSu/Breadcrumb-detailSU.png)

### Struktur Komponen

`Breadcrumb` memiliki struktur komponen sebagai berikut:

| Library / Modul | Komponen yang digunakan | Properti/Atribut | Tipe Data <br/> Atribut | Penjelasan                                                                                                                |
| --------------- | ----------------------- | ---------------- | ----------------------- | ------------------------------------------------------------------------------------------------------------------------- |
| `next/link`     | `<Link>`                | `-`              |                         | `<Link>` Merupakan library bawaan dari next js <br/> digunakan untuk menuju halaman yang akan dimasukkan di atribut href. |
|                 |                         | `href`           |                         | `href` Merupakan atribut yang ada di dalam `<Link>` <br/> yang berfungsi untuk menaruh alamat halaman yang ingin dituju.  |
