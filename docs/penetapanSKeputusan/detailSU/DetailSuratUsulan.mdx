# Detail Surat Usulan
Komponen Detail Surat Usulan untuk melihat detail data surat usulan dengan cara klik button detail.<br/>
Halaman ini menggunakan **`React-Bootstrap`** sebagai framework tampilan. Dan ada beberapa custom-component yang digunakan seperti `Breadcrumb` yang digunakan untuk menunjukkan hirarki navigasi,
`Form` yang digunakan untuk membuat komponen form, serta `Layout` yang merupakan tata letak dasar halaman.
